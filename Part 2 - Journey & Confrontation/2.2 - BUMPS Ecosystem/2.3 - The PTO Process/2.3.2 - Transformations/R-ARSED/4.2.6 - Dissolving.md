# Dissolving (D)

## Diagnostic Question

**“What would need to change so this problem could no longer exist?”**

---

## Definition

Dissolving removes a problem entirely by **redesigning the system or environment** so the conditions that produce the problem disappear.

> The problem becomes irrelevant.

---

## Essence

**Change the game.**

Dissolving does not fight the problem. It makes the problem impossible.

---

## Key Characteristics

* 🧨 Structural redesign
* 🏗️ System-level change
* 🚀 One-time transformation
* ❌ Problem elimination

---

## Typical Signals

* “That issue no longer exists”
* “We redesigned the system”
* “The problem doesn’t apply anymore”

---

## Examples

* Digital distribution dissolving piracy of physical media
* Seatbelts dissolving the problem of dashboard injuries
* Renewable energy dissolving fossil-fuel dependency

---

## Counter-Plays

* **Against Revolving:** Remove incentives that benefit from persistence.
* **Against Endless Solving:** Step back and redesign the environment.

---

## System Effect

The original problem disappears because **the system that produced it no longer exists**.
