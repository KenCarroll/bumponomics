# Agents & Actors
Agent based models are used to simulate the behavior of complex systems. 

- Agents: Autonomous entities (people, cars, cells) with internal states and behaviors.
- Environment: The space or context where agents exist, interact, and are influenced.
- Rules: Simple instructions governing agent actions and decisions (e.g., move if close to a predator).
- Interactions: How agents affect each other and their environment.
- Emergence: Complex patterns (like traffic jams, flocking birds, market trends) arising from simple, local interactions.
